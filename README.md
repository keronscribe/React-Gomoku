# React-Gomoku

## 連結
[五子棋小遊戲](https://keronscribe.github.io/React-Gomoku/dist/index.html)


## 介紹
簡易的五子棋遊戲程式。


這是 [Lidemy 程式導師實驗計畫](https://github.com/Lidemy/mentor-program-3rd)第 21 週的第二份作業，同時是我的第二個 React 作品。

## 功能
- 五子棋遊戲
- 可檢視先前棋譜


## 技術
- React
- React Hook
- CSS grid

## 環境
- 用 Webpack 架設 React 開發環境
